    HTML
    Sua finalidade é estruturar o conteúdo de uma página web, determinando como os elementos como texto, imagens, links e outros conteúdos devem ser organizados e exibidos no navegador
    CSS
    Sua finalidade é definir a apresentação visual de uma página web, separando o conteúdo (HTML) da sua forma de apresentação. Ele controla aspectos como cores, fontes, espaçamentos, layouts e animações, tornando as páginas mais atraentes e personalizadas.
    JAVASCRIPT
    A principal finalidade do JavaScript é tornar páginas web dinâmicas e interativas. Em vez de serem meramente estáticas, o JavaScript permite que as páginas respondam às ações do usuário, alterem seu conteúdo, e executem várias outras funcionalidades complexas, como animações, gráficos, e interações com servidores.
